Common Issues

Connection Issues

  • Cannot connect to Anki

    • Ensure Anki is running
    • Verify AnkiConnect is installed
    • Restart Anki and try again

  • AnkiConnect add-on not working

    • Check add-on code: 2055492159
    • Restart Anki after installation
    • Update to latest version

Processing Issues

  • No flashcards created

    • Check PDF formatting
    • Verify processing mode
    • Ensure page dimensions match

  • Error processing PDF

    • Check file permissions
    • Ensure PDF isn't corrupted
    • Try re-exporting the PDF

Troubleshooting Guide

Basic Checks

  • Ensure you're using the latest version of NotesAnkify
  • Verify Anki is running before starting NotesAnkify
  • Check if AnkiConnect add-on is properly installed
  • Confirm you have necessary permissions for file access

Log Files

Log files are saved in the following locations:

  • Windows:

    %USERPROFILE%\notesankify-logs\

  • macOS:

    ~/notesankify-logs/

  • Linux:

    ~/notesankify-logs/

Include these logs when reporting issues to help us better understand the problem.

Known Issues

  • Windows Security Warning

    When running NotesAnkify for the first time, Windows might show a security warning. Click "More info" and then "Run anyway" to proceed.

  • macOS Gatekeeper

    On macOS, you might need to right-click the app and select "Open" on first launch due to Gatekeeper security.
