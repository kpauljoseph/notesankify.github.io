Getting Started

Download and Install Prerequisites

First, ensure you have the required software installed:

  • Download Anki - Install the Anki desktop application
  • Install the AnkiConnect add-on:
    1. Open Anki
    2. Go to Tools → Add-ons → Get Add-ons...
    3. Enter code: 2055492159
    4. Restart Anki after installation
  • Download NotesAnkify for your operating system

Prepare Your Notes

NotesAnkify supports multiple ways to create flashcards:

Method 1: Using Markers

Add "QUESTION" and "ANSWER" text markers to your notes:

QUESTION
What is photosynthesis?

ANSWER
Process where plants convert...

Method 2: Standard Size

Use standard flashcard dimensions:

  • Width: 455.04 points
  • Height: 587.52 points
  • Question on top half
  • Answer on bottom half

Process Your Notes

Converting your notes is simple:

  1. Start Anki (must be running for NotesAnkify to work)
  2. Launch NotesAnkify
  3. Click "Browse" and select your PDF directory
  4. Choose your preferred processing mode:
    • Check both markers and dimensions
    • Only check for markers
    • Only check dimensions
    • Process all pages
  5. Click "Process and Send to Anki"

Start Studying!

Your flashcards are now ready:

  • Open Anki to see your new flashcards
  • Cards are organized in decks based on PDF file structure
  • Begin studying with Anki's spaced repetition system

  3. Run NotesAnkify.exe

Frequently Asked Questions

How do I format my PDF notes for NotesAnkify?

You have three options:

  • Add "QUESTION" and "ANSWER" markers in your notes
  • Use standard flashcard dimensions (455.04 × 587.52 points)
  • Split any page into top (question) and bottom (answer) halves

Does it work with handwritten notes?

Yes! NotesAnkify works with any PDF content, including handwritten notes, typed text, images, or diagrams. Just ensure your notes follow one of the supported formats.

Why do I need to install AnkiConnect?

AnkiConnect is required to allow NotesAnkify to communicate with Anki and automatically create flashcards. It's a secure add-on that acts as a bridge between NotesAnkify and Anki.

Will my existing Anki flashcards be affected?

No. NotesAnkify safely adds new flashcards to your specified decks without modifying existing ones. It also prevents duplicates to maintain a clean study collection.

What note-taking apps are supported?

NotesAnkify works with any app that can export to PDF, including:

  • GoodNotes
  • Notability
  • OneNote (exported as PDF)
  • Any other app that creates PDF files
