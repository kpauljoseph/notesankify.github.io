NotesAnkify Documentation

Installation

NotesAnkify is available for Windows, macOS, and Linux. Follow these steps to get started:

Windows

  1. Download the ZIP file for your system (AMD64/ARM64)
  2. Extract the ZIP file to your desired location
  3. Run NotesAnkify.exe
  4. On first run, Windows might show a security warning. Click "More info" and then "Run anyway"

macOS

  1. Download the DMG file
  2. Open the DMG file
  3. Drag NotesAnkify to your Applications folder
  4. Control-click the app and select "Open" on first launch
  5. Click "Open" when prompted about unverified developer

Linux

  1. Download the tar.xz file
  2. Extract using terminal: tar xf NotesAnkify-linux-*.tar.xz
  3. Make the file executable: chmod +x NotesAnkify
  4. Run the NotesAnkify executable

Prerequisites

Before using NotesAnkify, ensure you have the following installed:

Required Software

  • Anki Desktop:
    • Download from apps.ankiweb.net
    • Version 2.1.54 or later recommended
    • Must be running while using NotesAnkify
  • AnkiConnect Add-on:
    • Open Anki → Tools → Add-ons → Get Add-ons...
    • Enter code: 2055492159
    • Restart Anki after installation

System Requirements

  • Windows 10/11 (64-bit) or macOS 11+ or Linux
  • 4GB RAM recommended
  • 500MB free disk space
  • Internet connection for Anki sync (optional)

Quick Start Guide

Follow these steps to convert your first PDF notes to Anki flashcards:

Basic Usage

  1. Start Anki Desktop application
  2. Launch NotesAnkify
  3. Click "Browse" and select your PDF directory
  4. Choose a root deck name (optional)
  5. Select your preferred processing mode
  6. Click "Process and Send to Anki"

💡 Pro Tip:

Start with a small set of PDFs to familiarize yourself with the conversion process and ensure your notes are formatted correctly.

Formatting Your Notes

NotesAnkify supports multiple ways to format your notes for flashcard conversion:

Method 1: Using Markers

QUESTION
What is the capital of France?

ANSWER
Paris is the capital of France.

  • Add "QUESTION" text above your question
  • Add "ANSWER" text above your answer
  • Case sensitive - must be uppercase
  • Works with any page size

Method 2: Standard Dimensions

Create pages with standard flashcard dimensions:

  • Width: 455.04 points
  • Height: 587.52 points
  • Question goes on top half
  • Answer goes on bottom half
  • No markers needed

Method 3: Process All Pages

For simpler processing:

  • Any page size works
  • Top half becomes question
  • Bottom half becomes answer
  • No special formatting needed

Processing Modes

NotesAnkify offers four different processing modes to handle various note formats:

1. Markers and Dimensions Mode

Most Strict

  • Requires QUESTION/ANSWER markers
  • Must match standard dimensions
  • Best for consistent flashcard creation

2. Markers Only Mode

Flexible Dimensions

  • Only checks for QUESTION/ANSWER markers
  • Any page size accepted
  • Good for mixed-size documents

3. Dimensions Only Mode

Standard Size

  • Only checks page dimensions
  • Splits page into top/bottom
  • No markers needed

4. Process All Pages Mode

Most Flexible

  • Processes every page
  • Splits each page in half
  • No formatting requirements

Deck Organization

NotesAnkify organizes your flashcards into Anki decks based on your PDF file structure:

Deck Naming

  • Root Deck (Optional):
    • Specified in the app interface
    • All cards are placed under this deck
    • Example: "University" or "Study"
  • Subdeck Structure:
    • Follows your folder structure
    • Uses "::" as separator
    • Example: "University::Biology::Chapter1"

Example File Structure:

PDFs/
  └── Biology/
      ├── Chapter1.pdf
      └── Chapter2.pdf
  └── Chemistry/
      └── Notes.pdf

Results in decks:

  • Biology::Chapter1
  • Biology::Chapter2
  • Chemistry::Notes

Advanced Features

Output Directory

Specify a custom output directory to:

  • Save processed flashcard images
  • Review conversion results
  • Debug processing issues

Duplicate Detection

NotesAnkify prevents duplicate flashcards by:

  • Generating unique content hashes
  • Checking existing cards
  • Skipping identical content

Processing Report

After processing, you'll receive a detailed report showing:

  • Total PDFs processed
  • Number of flashcards created
  • Skipped (duplicate) cards
  • Processing time
  • Log file location

Common Issues

Connection Issues

Error: "Could not connect to Anki"

  1. Ensure Anki is running
  2. Check if AnkiConnect is installed
  3. Restart Anki and try again

Processing Issues

No flashcards created

  • Verify PDF formatting
  • Check chosen processing mode
  • Review processing report

Permission Issues

Access Denied Errors

  • Run as administrator (Windows)
  • Check folder permissions
  • Verify write access to output directory

Error Messages

Common error messages and their solutions:

"Failed to extract image from PDF"

  • Ensure PDF is not corrupted
  • Check if PDF is password protected
  • Try re-exporting the PDF

"Failed to create deck"

  • Check Anki is running
  • Verify AnkiConnect installation
  • Try a different deck name

"No PDF files found"

  • Check selected directory
  • Ensure files have .pdf extension
  • Verify file permissions

Frequently Asked Questions

Q: Can I modify cards after they're created?

A: Yes, you can edit any flashcard directly in Anki after creation.

Q: Will my cards sync to AnkiWeb?

A: Yes, cards will sync normally through Anki's built-in synchronization.

Q: Can I use custom card templates?

A: NotesAnkify uses its own template optimized for PDF content. You can modify cards after creation.

Q: Does it work with scanned PDFs?

A: Yes, NotesAnkify works with any PDF, including scanned documents.
